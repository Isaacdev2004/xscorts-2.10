IF MONGODB WON'T START:
-----------------------

Option 1: Check Windows Services
  1. Press Windows + R
  2. Type: services.msc
  3. Look for "MongoDB" service
  4. Right-click → Start

Option 2: Find MongoDB Installation
  1. Look in: C:\Program Files\MongoDB
  2. Find the version folder (like "Server\7.0\bin")
  3. Run: mongod.exe

Option 3: Reinstall MongoDB
  1. Make sure to check "Install as Service" during installation
  2. This makes it easier to start automatically
