================================================================================
                    WELCOME TO XSCORTS APPLICATION
================================================================================

QUICK START (3 SIMPLE STEPS):
-----------------------------

STEP 1: Install MongoDB
   → Go to: https://www.mongodb.com/try/download/community
   → Download and install (choose "Install as Service" during setup)
   → Double-click: START-MONGODB.bat

STEP 2: Start Redis  
   → Make sure Docker Desktop is running (if you have it)
   → Double-click: START-REDIS.bat
   → OR download Redis from: https://github.com/tporadowski/redis/releases

STEP 3: Start Application
   → Double-click: START-EVERYTHING.bat
   → Choose option 4 from the menu
   → Wait 30 seconds, then open http://localhost:8081

================================================================================

EASIEST WAY - USE THE MENU:
---------------------------
Just double-click: START-EVERYTHING.bat
Then follow the menu options!

================================================================================

WHAT YOU NEED:
-------------
✓ MongoDB (database) - Must install
✓ Redis (cache) - Use Docker OR install separately  
✓ Node.js - Should already be installed
✓ npm - Comes with Node.js

================================================================================

CHECK IF EVERYTHING IS WORKING:
-------------------------------
Double-click: CHECK-STATUS.bat

You should see:
  ✓ MongoDB is RUNNING
  ✓ Redis is RUNNING

If both show ✓, you're ready to start the application!
